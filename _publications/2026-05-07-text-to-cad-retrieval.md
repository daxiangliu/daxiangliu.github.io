## 中文

本工作关注 Text-to-CAD 跨模态检索：给定一段自然语言描述，从大规模 3D CAD 模型库中检索语义相关的 CAD 模型，为 CAD 模型复用和设计检索提供基础能力。

论文基于 Text2CAD 数据集建立任务基准，并提出一个统一框架，从 CAD 构造过程序列和几何点云中学习多模态 CAD 表示，再与文本查询表示进行跨模态匹配。该工作已作为 arXiv 预印本公开；代码链接公开后将补充到本页。

## English

This work studies Text-to-CAD cross-modal retrieval: given a natural language description, the goal is to retrieve semantically relevant CAD models from large-scale 3D CAD repositories, supporting CAD model reuse and design search.

The paper builds a benchmark based on the Text2CAD dataset and proposes a unified framework that learns multi-modal CAD representations from procedural sequences and geometric point clouds, then matches them with textual query representations. The work is now available as an arXiv preprint; the code link will be added here once it is public.
